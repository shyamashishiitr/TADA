## ğŸ§  The Geshido Method

TADA follows the Geshido productivity methodology:

1. **Capture Everything** - Use the Inbox for rapid task capture
2. **Process Daily** - Move tasks from Inbox to Today/This Week
3. **Execute** - Focus on Today's tasks
4. **Review Weekly** - Plan your week, move tasks from Someday
5. **Stay Minimal** - Keep Today's list short and achievable
